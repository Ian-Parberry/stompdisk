void GenerateShiShuaSeed ( uint64_t  seed[4])

Generate a pseudo-random shishua seed using the cstdlib function rand() seeded using timeGetTime(), the number of milliseconds since Windows last rebooted. Since rand() returns a 16-bit result, each 64-bit part of the seed is constructed using four calls to rand() shifted appropriately.

Parameters
seed[out] A shishua seed.
